# Report-Card-Generator
A beginner-friendly Python project that generates a report card from user input. The user enters their name and any number of subjects with marks, and the program calculates the total, average, and assigns a letter grade.
Features:

Accepts any name and any number of subjects
Calculates total and average marks
Assigns a grade (A–D) based on average

Concepts used: variables, loops, dictionaries, conditionals, user input (input())
